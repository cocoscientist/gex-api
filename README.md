## An API for Gex Quotes!
This API serves to return quotes from the game *Gex - Enter the Gecko!* <br />
Following is the format in which a quote is stored
```
    {
        "id":<Item ID>,
        "level":<Level Name>,
        "quote":<Main Quote>
    }
```
